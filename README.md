# Iqra Shahzadi - Resume

This repository contains the **HTML and CSS files** for my professional resume.  
It is designed to be **clean, simple, and ATS-friendly**, highlighting my skills, experience, projects, and education.

## Features

- Fully responsive layout that works on desktop and mobile screens
- Organized sections: Professional Objective, Technical Skills, Experience, Projects, Education, Additional Information
- Styled using modern CSS for readability and visual clarity
- Simple file structure for easy updates

## Technologies Used

- HTML5
- CSS3
- Modern web development best practices

## Project Structure

```bash
iqra-shahzadi-resume/
├─ index.html # Main resume page
├─ style.css # External stylesheet
└─ README.md # Project documentation
```
